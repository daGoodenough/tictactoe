The Project

        This seemed like it would be a good first real project because it would implement Javascript to define the game logic, and also basic css and html to make the game visually appealing.

        I got to use some jQuery to aid in making the website interactive. I was getting confused between DOM methods and jQuery methods, so my JS ended up being a mix of the two. Because of that, this will be something that I focus more on in the future... being consistent with the way I write my code.

        The hardest part of this project was definitely defining the logic for the game. I followed along with someone building tic-tac-toe using JS a few months ago, so that helped some but they did it completely in the console. Also, I could hardly remember what they did. I realized that it was more efficient to reason through it myself rather than trying to remember each step that they took. In the end, I think that my code may be longer than it has to be, but at least it works. I did a pretty good job about keeping all of my functions separate, but I do think that writing more pseudo-code in the beginning will help me stay organized.

        All in all, I am really proud of this as my first successful project. I say "first successful" because I started in on a portfolio site, but then I realized I had no projects to display so didn't have much content to work with. This project was a great learning experience for me, and I am glad that I took the time to do it; I think that I learned more from this than in the last 3 months of doing tutorials and watching videos. My biggest take away from the whole thing was that I just have to keep working and trying and eventually it will manifest itself. If I don't write any code, then nothing happens.
